## Purpose

`@luminable/stack-backend` provides a basic backend setup for Luminable.

## Shared Repository Guidelines (Required)

All edits in this package must follow the shared repository guidelines in `../../docs/GUIDELINES.md`.

## Do / Don't

- Keep exports explicit and stable.
- Keep encapsulate common backend conventions to serve as a sensible default.
- Don't restrict consumers from replacing or extending the stack configuration.
