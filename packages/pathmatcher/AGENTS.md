# AGENTS Guide: @luminable/pathmatcher

## Purpose

`@luminable/pathmatcher` provides path matching utils for pathname-based routing.

## Shared Repository Guidelines (Required)

All edits in this package must follow the shared repository guidelines in `../../docs/GUIDELINES.md`.

## Do / Don’t

- Keep exports explicit and stable.
- Keep the matching logic highly optimized and tested.
- Don't add external runtime dependencies unless absolutely necessary.
