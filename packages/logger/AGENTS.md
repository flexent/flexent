# AGENTS Guide: @luminable/logger

## Purpose

`@luminable/logger` provides an idiomatic logger API supporting custom formatting and transports.

## Shared Repository Guidelines (Required)

All edits in this package must follow the shared repository guidelines in `../../docs/GUIDELINES.md`.

## Do / Don't

- Do keep exports explicit and stable.
- Do make logger transports easily pluggable.
- Don’t add external runtime dependencies unless absolutely necessary.
